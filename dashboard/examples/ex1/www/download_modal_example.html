What is included in this data download?

This file contains the data behind many of the indicators displayed through this portal. We are unable to make all datasets available in this file.

The data is presented in a long-table csv format for easy filtering.

Columns included

  • class: The indicator class, such as economic, social or health
  • category: The indicator category, such as activity or confidence
  • indicator_name: The indicator name
  • series_name : The series name
  • sub_series_name : The sub-series name (from series with multiple lines)
  • date : The date (in the format dd/mm/yyyy)
  • value: The value of the series
  • units: The units associated with this series.
  • last_update_date: The date we last updated this indicator (not the last date in the series).
